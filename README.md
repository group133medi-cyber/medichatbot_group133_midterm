This Repository is to showcase our progress during the project mid term (By week 4).
We had developed a UI code and a Chatbot code in two different machines.
To combine them we used ngrok as a medium.
Ngrok provided us an public Url to connect our UI and Chatbot codeinto a local development server.
This caused the challenge of needing both UI code and the chatbot code to run together and always having network connection on both machines to make chatbot work
Due to this limitation we are working on having both codes work on cloud.
